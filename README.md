# 🚀 AI Career Navigator (Capstone Project)

This project is an AI-powered career guidance system designed to help students choose the right career path based on their skills, interests, and performance.

---

## 📌 Features

- 🤖 AI Chatbot for career guidance  
- 📊 Career recommendation system  
- 🧠 Skill gap analysis  
- 📝 Aptitude & interest-based tests  
- 🎓 College search and suggestions  
- 👤 Student dashboard  
- 🛠 Admin panel  

---

## 🛠 Tech Stack

- Frontend: React.js  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Authentication: JWT  
- AI/NLP Integration
